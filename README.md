## 🚀 Quick Start

Follow these steps to run the cosmos on your local machine.

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)
- PostgreSQL (if local DB is used)
- Redis (for caching)

### 1. Backend Setup
```bash
cd astrology-backend
python -m venv venv
source venv/Scripts/activate  # On Windows
pip install -r requirements.txt
python main.py
```

### 2. Frontend Setup
```bash
cd astrology-frontend
npm install
npm run dev
```
